GlobalApp vs. Mini App

the globalApp is a full blown library, its purpose is to supply the miniSPAs with directives. each directive is a component that can be used in the templating system. The main goal of the globalApp is to force the code to be global. for instance an "input" field is a directive, and the instence of the global directive is plced on a miniSpa page. When we update the "input" directive the entire solution (all miniSPAs) should update.
